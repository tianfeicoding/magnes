## 装饰元素

### Washi 胶带
- 位置：卡片边缘，斜向贴出
- 尺寸：宽 45-60px，高 18px
- 图案：斜条纹（赭石 + 米黄交替）
- 透明度：0.7

### 角落 Emoji
- 尺寸：14-24px
- 透明度：0.25-0.6
- 位置：四角及边缘
- 类型：与主题相关（面包用 🥐🍞，咖啡用 ☕🥛）

### 手绘边框
- 卡片外框：2px solid 灰绿
- 虚线装饰：1px dashed 赭石（卡片外层）
- 阴影：3px 3px 0px 灰绿（手绘感偏移）

## 截图指南

1. 浏览器打开 HTML 文件
2. F12 打开开发者工具
3. 点击 Device Toolbar（手机图标）
4. 选择 Responsive，设置尺寸为 375x500
5. 右键 → 截图节点（Screenshot node）
6. 保存为 PNG，上传小红书

## 常见错误

- ❌ 使用 min-height 导致内容撑高 → 改用固定 height
- ❌ 字体过大导致卡片拥挤 → 主标题不超过 22px
- ❌ 条目超过 5 个 → 建议 3-4 个最佳
- ❌ 装饰元素遮挡文字 → 调整 z-index 和位置
- ❌ 使用真实图片 → 建议用 emoji + CSS 装饰，避免加载问题
