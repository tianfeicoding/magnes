📋 节点架构说明

Rednote 模块采用三节点架构，每个节点负责不同的功能：

┌─────────────────┐ ┌─────────────────┐ ┌─────────────────┐ │ Content Node │ ───▶ │ StyleLab Node │ ───▶ │ Preview Node │ │ 内容输入节点 │ │ 风格实验室 │ │ 预览节点 │ └─────────────────┘ └─────────────────┘ └─────────────────┘ │ │ │ ▼ ▼ ▼ 文本 + 图片 风格配置 + 模板 小红书界面预览 + AI 功能

工作流程：
1. Content Node 接收用户输入的文本和图片
2. StyleLab Node 应用风格配置和选择模板
3. Preview Node 在手机界面中预览最终效果
4. AI 功能 提供智能换背景、智能排版和智能配色
